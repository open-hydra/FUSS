Everything you need for solid heat transfer

A complete, validated thermal solver written in modern Fortran

Solid Heat Conduction

Transient and steady-state Fourier heat equation on multi-block structured grids with temperature-dependent material properties.

Cell-Centred Finite Volume

Second-order central differences for the diffusive flux. Arithmetic-mean conductivity within materials; harmonic-mean across interfaces.

Mixed Boundary Conditions

Fixed temperature (constant or time-varying), prescribed flux, convection, radiation, symmetry, and block-to-block connections — applicable face-by-face.

Accelerated Time Marching

3-stage SSP Runge–Kutta with implicit residual smoothing and geometric multigrid. Local time stepping for fast steady-state convergence.

Parallel Execution

Shared-memory parallelism via OpenMP and distributed-memory MPI with ghost-cell exchange across block boundaries on HPC clusters.

Flexible I/O

Tecplot (ASCII & binary) and VTK output. Point probes for time histories and full restart capability.

Up and running in minutes

FUSS ships with a single install script that handles all dependencies, compiles the Fortran core, and puts the binary on your PATH.

Installation guide → Quick start
terminal 
# Clone and build
$ git clone https://github.com/open-hydra/FUSS.git
$ cd FUSS && ./install.sh build --compilers=gnu

# Run a simulation
$ cd test/steady-state/temperature_Plate
$ ./FUSS.sh solve
